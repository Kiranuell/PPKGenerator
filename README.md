# PPKGenerator
Небольшой код для генерации презентаций для "PowerPoint караоке"

# Принцип работы
В папке *assets* должны находится:
* текстовые файлы с наборами вводных фраз, каждая на новой строчке
* папки с наборами изображений на определенные темы

В *config* файле имеется структура создаваемой презентации
Первая буква тип слайда:
* t - случайная фраза на весь слайд
* i - случайное изображение на весь слайд

Через пробел идет название файла или папки откуда будет браться содержание

Итоговая презентация после работы кода сохраняется в *output.pdf* (pptx формат отвратительный)
